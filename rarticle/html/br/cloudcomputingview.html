Cloud Computing - Minha primeira impressão

Olá, hoje eu gostaria de falar sobre Cloud Computing.

Acabei de entrar nesse mundo, claro, como um DevOps engineer e não como um usuário já que a computação em nuvem está aí há um bom tempo.

Comecei lendo sobre AWS, mas logo um serviço chamado S3 me saltou aos olhos.

Muito resumidamente, o serviço S3 nada mais é do que espaço de dados na nuvem.

Você pode criar, copiar, remover e apagar conteúdo do seu computador para o S3 e do S3 para o seu computador.

Bom, isso tudo é muito interessante ... mas o que realmente me chamou a atenção foi o fato de que usar a nuvem é muito parecido com usar um ambiente local.

Por exemple, para copiar um arquivo da minha máquina para o S3 eu só preciso executar um comando: "aws s3 cp".

Algo assim: aws s3 cp file.txt s3://ronaldoafonso

Para o processo inverso eu executo: aws s3 cp s3://ronaldoafonso/file.txt file.txt

Simples assim.

Não lhe parece aquele antigo comando "cp" que você executou pela primeira vez quando estava aprendendo o Linux shell?

Para mim parece muito.

Isso é tudo.

Eu sei que é um artigo curto, mas a minha intensão aqui é somente fazer você parar para pensar sobre onde a nuvem está nos levando.
