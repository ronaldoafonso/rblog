Yocto - Série Linux Embedded 1.

Olá, hoje vou começar a falar sobre o projeto Yocto.

O projeto Yocto se define como "não sendo uma distribuição para Linux embarcado, mas uma que cria uma distrubuição Linux customizada para você".

Vou falar sobre esta definição e mais outras coisas ao longo desta serie de artigos, mas por enquanto vamos começar pelo básico.

Antes de mais nada, quais componentes são necessários para uma distribuição de Linux Embarcado?

Veja a list a seguir:

  • Toolchain: uma coleção de ferramentas usadas na criação (compilação, link, etc) dos binários que irão rodar na sua distribuição Linux.
  • Boot Loader: o programa que carrega o sistema Linux no seu dispositivo.
  • Kernel: é o componente principal de um sistema Linux.
  • Root Filesystem: todos os arquivos necessários para que um sistema Linux possa operar e fazer o seu trabalho.

Isso é tudo por hora.

Eu falarei mais sobre estes tópicos em artigos futuros.
