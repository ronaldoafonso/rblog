Yocto - Série Linux Embarcado 2.

Olá

No artigo anterior eu falei sobre os componentes de um sistema Linux embarcado.

Agora é hora de falar sobre o projeto Yocto.

Então, o que é o projeto Yocto?

Segundo o site Wikipedia, Yocto é o menor prefixo de medida do sistema internacional de medidas.

Ok, mas aqui estamos falando do projeto open source Yocto, ou seja, um projeto com a finalidade de nos ajudar a criar um sistema Linux customizado para um dispositivo embarcado.

O projeto Yocto nos fornece ferramentas para facilitar nossas vidas como desenvolvedores para Linux embarcado quando precisamos colocar todos os componentes de um sistema Linux juntos. Por exemplo: um toolchain, um bootloader, um kernel e um root filesystem.

A coisa funciona basicamente assim: nós fornecemos ao projeto Yocto, através de suas ferramentas, aquilo que queremos, como: uma configuração de kernel, a arquitetura do nosso hardware, a list de binários, etc ... e o projeto Yocto nos devolve um sistema Linux que pode ser instalado em nosso dispositivo.

Isso será o que faremos durante esta série de artigos.

Ao final nós deveremos ter um dispositivo rodando o nosso sistema Linux customizado.
