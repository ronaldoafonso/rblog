Yocto - Linux Integrado Serie 2.

Hola.

En lo artícolo anterior he hablado sobre los componentes de un sistema Linux.

Ahora es el momento de hablar sobre el proyecto Yocto en sí.

Entonces, ¿Qué es el proyecto Yocto?

Según Wikipedia, Yocto es el prefijo de unidad más pequeño del sistema métrico.

Ok, pero aquí estamos hablando del proyecto Yocto de código abierto, es decir, un proyecto que pretende ayudarnos a crear un sistema personalizado basado en Linux para nuestro dispositivo integrado.

El proyecto Yocto proporciona herramientas que nos facilitan la vida como desarrolladores de Linux integrado cuando tenemos que reunir todos los componentes de un sistema Linux integrado, como: un toolchain, un bootloader, un kernel y un root filesystem.

La cosa básicamente funciona así: informamos al proyecto Yocto, a través de sus herramientas, lo que queremos, como: la configuración del kernel, la arquitectura hardware de nuestro dispositivo, la lista de binarios, etc ... y el proyecto Yocto nos devuelve un sistema Linux que podemos instalar en nuestro dispositivo.

Eso es lo que haremos durante toda esta serie de artículos.

Al final deberíamos tener un dispositivo que funcione con nuestro sistema Linux personalizado.
