Yocto - Linux Integrado Serie 1.

Hola.

Hoy comienzo a hablar del proyecto Yocto.

El proyecto Yocto se define a sí mismo como "no una distribución de Linux integrado, sino una que crea una distribución de Linux personalizada para usted".

Bueno, hablaré sobre esa definicíon y mucho más a lo largo de esta serie de artícolos, pero ahora comencemos con lo básico.

Ante todo, que componentes una distribución Linux deve tener?

Consulte la siguiente lista:

  • Toolchain: una colección de herramientas que se utilizan para crear (compilar, vincular, etc.) los archivos binarios que se ejecutarán en su distribución de Linux de destino.
  • Boot Loader: un programa que carga el sistema Linux en un dispositivo de destino.
  • Kernel: lo principal componente de un sistema Linux.
  • Root Filesystem: todos los archivos que un sistema Linux necesita para operar o hacer su trabajo.
