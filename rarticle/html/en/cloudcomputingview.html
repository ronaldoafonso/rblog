Cloud Computing - My First Impression

Hi, I just got started in the Cloud Computing world!

Well, of course, as a DevOps engineer not as a user of the cloud since it has been out there for a long time.

I started reading about AWS, and soon I came across a service known as S3.

Long story short, S3 is storage in the cloud.

You can create, copy, move, and delete data from your computer to S3 and the other way around.

So, it's all very interesting ... but what really got my attention was the fact that when you're using the cloud it looks pretty much as if you were doing things in a local environment.

For example, to copy a file from my computer to S3 I just need to run an: "aws s3 cp" command.

Check it out: aws s3 cp file.txt s3://ronaldoafonso

In order to get the file back I do: aws s3 cp s3://ronaldoafonso/file.txt file.txt

Just like that.

Doesn't it look like an old "cp" command you executed when you were first introduced to the Linux shell?

Well, it definitely does to me.

That is it.

I know it's a short article, but my intention here is just to make you think about where we're heading in the cloud.
