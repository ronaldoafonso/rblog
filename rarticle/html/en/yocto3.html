Now, it's time to talk about setting up a build machine.

So, what is a build machine or what is it used for?

To build an image of your customized Linux system, you'll need another Linux system.

Well, you need a "traditional" Linux distribution where you can download the Yocto project, so it can build a customized Linux system for you.

I'm going to use a Debian 10 "buster" here, but there are other Linux distributions you can use as your build machine.

At the current time, the prerequisites for a build machine are:

  • 50 Gbytes of free disk space.
  • Git 1.8.3.1 or greater.
  • tar 1.28 or greater.
  • Python 3.50 or greater.
  • gcc 5.0 or greater.

Depending on your Linux distribution, you'll need a certain list of packages installed.

In Debian 10 case, it would be:

apt-get install gawk wget git-core diffstat unzip texinfo gcc-multilib build-essential chrpath socat cpio python3 python3-pip python3-pexpect xz-utils debianutils iputils-ping python3-git python3-jinja2 libegl1-mesa libsdl1.2-dev pylint3 xterm python3-subunit mesa-common-dev

That's it.
