Ora è il momento di parlare di una build machine.

Ma cos'è una build machine?

Per creare un'immagine del tuo sistema Linux personalizzato, avrai bisogno di un altro sistema Linux.

Bene, hai bisogno di una distribuzione Linux "tradizionale" dove puoi scaricare il progetto Yocto, in modo che possa costruire un sistema Linux personalizzato per te.

Userò un Debian 10 "buster" qui, ma ci sono altre distribuzioni Linux che puoi usare come build machine.

In questo momento, i prerequisiti per una build machine sono:

  • 50 Gbytes di spazio libero su disco.
  • Git 1.8.3.1 o superior.
  • tar 1.28 o superior.
  • Python 3.50 o superior.
  • gcc 5.0 o superior.

A seconda della tua distribuzione Linux, avrai bisogno di un certo elenco di pacchetti installati.

Nel caso di Debian 10, sarebbe:

apt-get install gawk wget git-core diffstat unzip texinfo gcc-multilib build-essential chrpath socat cpio python3 python3-pip python3-pexpect xz-utils debianutils iputils-ping python3-git python3-jinja2 libegl1-mesa libsdl1.2-dev pylint3 xterm python3-subunit mesa-common-dev

Questo è tutto.

Nel prossimo articolo parlerò di "poky", il software utilizzato per generare un sistema Linux personalizzato.
