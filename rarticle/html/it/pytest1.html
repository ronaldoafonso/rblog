Pytest - Articolo 1

Questo è il primo articolo sul framework Python più utilizzato per i test, il "pytest".

Oggi descriverò solo come installare "pytest" in un ambiente virtuale Python.

Nei prossimi articoli ti fornirò maggiori dettagli su come usarlo.

Avrai bisogno almeno di un'installazione Python sulla tua macchina per creare un ambiente virtuale.

La maggior parte delle distribuzioni Linux vengono fornite con Python già installato, ma se il tuo non lo fa, dai un'occhiata alla documentazione di gestione dei pacchetti e vedi come installarlo.

Userò qui la versione 3.8.2 di Python.

Quindi, per questo primo articolo segui questi passaggi:

Passaggio 1: scegli la directory principale del progetto.

Devi solo andare nella directory in cui risiederà il tuo progetto.

Il mio sarà a: "/home/ronaldo/src/calc".

Passaggio 2: crea l'ambiente virtuale

All'interno della directory del progetto, eseguire: python -m venv venv-3.8.2.

E poi: source ./venv-3.8.2/bin/activate.

Bene, il tuo ambiente virtuale è sia creato e anche attivato.

Passaggio 3: installa "pytest"

Ora che hai attivato il tuo ambiente virtuale, esegui: pip install pytest.

Forse quel comando richiederà un po' di tempo, ma non appena avrà finito il tuo pytest sarà pronto a fare il suo lavoro.

Per verificarlo, esegui: pytest --version.

Dovresti vedere qualcosa del tipo: pytest 6.1.1. Se è così, il tuo ambiente è pronto. Altrimenti, dai un'occhiata ai passaggi precedenti. Potresti esserti perso qualcosa.

Bene, per oggi è tutto. Nei prossimi articoli ho intenzione di mostrarvi come creare alcuni test e come utilizzare la funzione "mark" di "pytest".

Ci vediamo!
