Yocto - Serie Linux Incorporato 1.

Ciao. Oggi inizierò a parlare del progetto Yocto.

Il progetto Yocto si definisce come "non una distribuzione Linux incorporata, ma una che crea una distribuzione Linux personalizzata per te".

Bene, parlerò di quella definizione e molto di più in questa serie di articoli, ma per ora iniziamo con le basi.

Prima di tutto, quali componenti deve avere una distribuzione Linux incorporata?

Dai un'occhiata in questa lista:

  • Toolchain: una raccolta di strumenti utilizzati per creare (compilare, collegare, ecc.) i binari che verranno eseguiti sulla distribuzione Linux di destinazione.
  • Boot Loader: un programma che carica il sistema Linux su un dispositivo di destinazione.
  • Kernel: il componente principale del sistema Linux.
  • Root Filesystem: tutti i file di cui sistema Linux necessita per funzionare o per svolgere il proprio lavoro.

È tutto per oggi.
