Pytest - Artículo 1

Esto és lo primero artículo acerca de lo framework para las pruebas mas utilizado en Python, el "pytest".

Hoy solo voy a describir cómo instalarlo en un entorno virtual de Python.

En los próximos artículos te daré más detalhes sobre cómo usarlo.

Necesitará al meno una instalación de Python en su máquina para crear un entorno virtual.

La mayoría de las distribuciones de Linux se envían con Python ya instalado, pero si la suya no lo tiene, consulte la documentación de administración de paquetes y vea cómo instalarlo.

Voy a usar la versión 3.8.2 de Python aquí.

Puedes encontrar todo el código del proyecto en: rcalc.

Entonces, para este primero artículo, simplemente siga estos pasos:

Paso 1: Elija el directorio principal de su proyecto.

Solo necesita ir al directorio donde residirá su proyecto.

El mío estará en: "/home/ronaldo/src/calc".

Paso 2: Cree el entorno virtual

Dentro del directorio del proyecto, ejecute: python -m venv venv-3.8.2.

Y luego: source ./venv-3.8.2/bin/activate.

Bien, su entorno virtual está creado y activado.

Paso 3: Instale "pytest"

Ahora que tiene su entorno virtual activado, ejecute: pip install pytest.

Talvez ese comando lleve algo de tiempo, pero tan pronto como haya terminado, su "pytest" estará listo.

Para verificarlo, ejecute: pytest --version.

Debería ver algo como: pytest 6.1.1. Si así es, su entorno está listo. De lo contrario, vuelva a mirar los pasos anteriores.

Bueno, eso es todo por hoy. En los próximos artículos voy a mostrarles cómo crear algunas pruebas y cómo usar la función "mark" de "pytest".
