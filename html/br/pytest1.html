Pytest - Artigo 1

Este é o primeiro artigo sobre o framework de testes mais usado em Python, "pytest".

Hoje só vou descrever como criar um ambiente virtual e instalar o "pytest".

Aguarde outros artivos onde descrevo com mais detalhes como o utilizamos.

Para iniciar o ambiente virtual, você precisa pelo menos ter uma instalação de Python na sua máquina.

A maioria das distros Linux já vem com o Python instalado. Se por algum motivo a sua é diferente, dê uma olhada no seu gerenciador de pacotes e veja como instalar o Python.

Eu vou usar aqui o Python versão 3.8.2.

Você pode clonar o projeto "rcalc" em: rcalc.

Para este primeiro artigo, você deve simplesmente executar os seguintes passos:

Passo 1: Escolher o diretório do projeto

Simplesmente vá até o diretório onde deseja que fique o seu projeto.

O meu será "/home/ronaldo/src/calc".

Passo 2: Criar o ambiente virtual

Dentro do diretório do projeto execute: python -m venv venv-3.8.2.

Feito isso, execute: source ./venv-3.8.2/bin/activate.

Pronto, o seu ambiente virtual está criado e ativado.

Passo 3: Instalar o "pytest"

Com o ambiente virtual já ativado, execute: pip install pytest.

Esse commando deve ser um pouco mais demorado, mas assim que terminar você já terá um ambiente "pytest" pronto para ser usado.

Para verificar se tudo está ok, execute: pytest --version.

Se você vir algo como: pytest 6.1.1, significa se o seu ambiente está pronto. Caso contrário, verifique novamente se você seguiu corretamente os passos anteriores.

Por enquanto é só isto. No próximo artigo vou mostrar como criar alguns testes e usar o recurso de "mark" de testes do "pytest".
