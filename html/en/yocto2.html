Now, it's time to talk about the Yocto project itself.

So, what is the Yocto project?

According to Wikipedia, Yocto is the smallest unit prefix in the metric system.

Ok, but here we're talking about the open source Yocto project, that is, a project intented to help us create a custom Linux-based system for our embedded device.

The Yocto project provides tools that makes our lives as embedded Linux developers easier when we have to put together all the compoments of an embedded Linux system, like: a toolchain, a bootloader, a kernel, and a root filesystem.

The thing basically works like this: we inform the Yocto project, through its tools, what we want, like: the kernel configuraion, the hardware architecture of our device, the list of binaries, etc ... and the Yocto project gives us back a Linux system that we can install in our target device.

That's what we'll be doing during all this series of articles.

At the end we should have a working device with our customized Linux system.
